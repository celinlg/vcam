# 🔥 TITAN ADB PRO ULTRA - V5

Uma interface gráfica moderna e completa para controlar dispositivos Android via ADB.

## 📋 Recursos

✅ **Explorador de Arquivos** - Navegue pelo Android
✅ **App Launcher** - Inicie aplicativos
✅ **Gerenciador de Apps** - Instale/Desinstale APKs
✅ **Apps Rodando** - Veja processos em execução
✅ **Monitor** - Monitore CPU e recursos
✅ **DSU Manager** - Gerencie imagens DSU
✅ **Shell ADB** - Execute comandos shell
✅ **Espelhamento** - Controle remoto via Scrcpy

## 🚀 Instalação

```bash
pip install -r requirements.txt
python main.py
```
